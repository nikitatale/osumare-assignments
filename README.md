# Osumare Assignments

This repository contains two assignments for Osumare Marketing Solutions:

1. **Frontend Assignment** – Responsive Internal Landing Page  
2. **Backend Assignment** – Basic RESTful API with Node.js and Express


## Frontend Assignment

**Objective:**  
Build a responsive landing page based on the provided Figma design.

**Requirements:**  
- Responsive design (mobile, tablet, desktop)  
- Pixel-perfect implementation  
- Cross-browser compatibility  
- No AI-copied code

## Backend Assignment

**Objective:**  
Build a RESTful API to manage tasks (to-do items) with CRUD operations.

**Endpoints:**  
- `GET /tasks` – list all tasks  
- `GET /tasks/:id` – get task by ID  
- `POST /tasks` – create task  
- `PUT /tasks/:id` – update task  
- `DELETE /tasks/:id` – delete task  

**Requirements:**  
- Use Express.js  
- Basic validation (`title`, `description`)  
- Store tasks in memory  
- Proper status codes (200, 400, 404)  
- Error handling  
- No AI-copied code
